# 📩 Spam SMS Detection using Machine Learning

## 📌 Introduction

Spam SMS Detection is a Natural Language Processing (NLP) and Machine Learning project that classifies SMS messages as either **Spam** or **Ham (Not Spam)**. The project helps automate message filtering, reducing unwanted spam while improving communication security and user experience.

---

## 🎯 Objectives

- Classify SMS messages as Spam or Ham.
- Perform text preprocessing and feature extraction.
- Train and evaluate machine learning classification models.
- Compare model performance using standard evaluation metrics.
- Improve spam detection accuracy using NLP techniques.

---
